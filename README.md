# HTML Practice Tasks

This repository contains a collection of beginner HTML practice tasks created using Visual Studio Code.
The purpose of these tasks is to understand the basic structure of HTML and how different HTML elements are used to create webpages and forms.

---

## Task 1 – HTML5 Structure

Created a basic HTML5 webpage structure using semantic elements such as **header, nav, section, article, and footer**.
This task helps understand how HTML5 organizes webpage content in a structured and readable format.

---

## Task 2 – HTML Tags Practice

Used common HTML tags such as **headings, paragraphs, lists, links, and images** to learn how content is displayed on a webpage.

---

## Task 3 – Survey Form

Developed a survey form that allows users to provide feedback.
The form includes different input elements such as:

* Text input fields
* Radio buttons for single choice questions
* Checkboxes for selecting multiple options
* Dropdown menus
* Textarea for longer responses
* Rating scale from 1 to 5

---

## Task 4 – User Registration Form

Created a registration form where users can enter their personal information such as:

* First name
* Last name
* Email address
* Password and confirm password
* Date of birth
* Gender selection
* Country dropdown
* Checkbox to agree to terms and conditions
* Submit button

---

## Task 5 – Navigation Page

Created a main **index.html** page that acts as a navigation dashboard to access all the tasks easily.

---

## Task 6 – Form Validation Basics

Used HTML attributes such as **required, minlength, and type validation** to ensure users enter valid data in form fields.

---

## Task 7 – Project Organization

Organized all HTML files in a structured folder and deployed the project using GitHub Pages to make the webpages accessible online.

---

## Technologies Used

* HTML5
* Basic CSS
* Visual Studio Code
* GitHub

---

## Purpose of the Project

The goal of this project is to practice basic web development concepts and understand how HTML is used to build structured and interactive webpages.
