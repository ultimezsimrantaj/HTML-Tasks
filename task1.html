About HTML

HTML (HyperText Markup Language) is the standard language used to create webpages. It provides the structure of websites using elements and tags.

HTML5 Semantic Tags

  • header – Defines the top section of a webpage
  • nav – Used for navigation links
  • section – Groups related content
  • article – Independent content block
  • footer – Bottom section of a webpage
